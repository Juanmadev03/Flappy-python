# Flappy Bird Python

Juego tipo Flappy Bird desarrollado con Python y pygame.

## Controles

ESPACIO → saltar

## Características

* Física con gravedad
* Tubos aleatorios
* Sistema de puntuación
* Colisiones
* Game Over

## Tecnologías

Python
pygame

## Ejecutar

pip install -r requirements.txt
python flappy.py
